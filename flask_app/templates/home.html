Methods Used

1

Data Collection: Web scraping using BeautifulSoup and manual data entry from various sources.

2

Viewership Modeling: Two-step process using lasso models with 5-fold cross-validation in scikit-learn.

3

Scheduling: Integer linear optimization using Google's OR-TOOLS package in Python.

Intrigue Score Methodology

The Intrigue Score quantifies a team's appeal to viewers using several key factors:

Win Percentage: Recent success attracts more viewers
Twitter Followers: Social media engagement indicator
Jersey Sales: Popular players boost attractiveness
Market Size: Larger markets draw more attention
